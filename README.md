This project is a tool for automating participation in Telegram giveaways (GiveShare and similar contests) using multiple accounts, built with Python and Playwright.
The bot simulates real user behavior, allowing you to automatically participate in contests and manage the process across multiple Telegram accounts.
🔧 Features:
Multi-account support
Automatic joining of Telegram channels
Simulation of real user actions (to reduce detection risk)
Fast and scalable automation powered by Playwright
Easy setup and configuration
💡 Use cases:
Mass participation in Telegram giveaways

Данный проект представляет собой инструмент для автоматизации участия в Telegram-гиввеях (GiveShare и аналогичные конкурсы) с использованием нескольких аккаунтов, реализованный на Python и Playwright.
Бот имитирует действия реального пользователя, позволяя автоматически участвовать в конкурсах и управлять процессом с разных Telegram-аккаунтов.
🔧 Функционал:
Поддержка нескольких аккаунтов
Автоматическое вступление в Telegram-каналы
Имитация действий пользователя (для снижения риска обнаружения)
Быстрая и масштабируемая автоматизация на базе Playwright
Простая настройка и запуск
💡 Применение:
Массовое участие в Telegram-гиввеях
